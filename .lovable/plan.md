

# Studio Clock App

A professional broadcast-style studio clock with a sleek, dark interface featuring red LED-style digits and an outer seconds ring, plus an integrated stopwatch. Works on Mac, PC, and any device with a modern browser.

---

## Main Clock Display

- Large **digital time display** in 24-hour format (HH:MM:SS)
- Bright red LED-style digits on a dark/black background
- Clean, segmented display font for authentic studio clock look
- **Outer seconds ring** with 60 LED-style dots that progressively illuminate as seconds tick
- **Date display** beneath the time (e.g., "Monday, 27 January 2026")

---

## Stopwatch Feature

- Positioned below the main clock
- Start/Stop and Reset buttons
- Displays elapsed time in HH:MM:SS.ms format
- Same red LED styling to match the main clock

---

## Fullscreen Mode

- Button to enter fullscreen for dedicated display use
- Perfect for studio monitors or dedicated clock displays
- Clean interface with no distractions in fullscreen

---

## Cross-Platform Compatibility

- Works on Mac, Windows, and Linux
- Runs in any modern browser (Chrome, Firefox, Safari, Edge)
- Completely offline - uses computer's local system clock
- Responsive design that scales well on any screen size
- No installation required - just open in browser
